# MessagePusher

MessagePusher 是一个统一的消息推送服务，支持多种消息渠道和平台。

## 技术栈

- **编程语言**：Python
- **数据库**：SQLite
- **Web框架**：Flask/FastAPI
- **前端**：简单的HTML/CSS/JavaScript
- **部署**：Docker
- **任务调度**：APScheduler
